# Custom Language Compiler

A complete compiler for a custom imperative programming language that compiles to JVM bytecode. The compiler implements all phases of compilation: lexical analysis, parsing, semantic analysis, and code generation.

## Features

### Core Language Features
- **Variables & Types**: INT, FLOAT, STRING, BOOLEAN, user-defined collections
- **Control Flow**: if/else, while loops, for loops
- **Functions**: User-defined functions with parameters and return values
- **Arrays**: Dynamic arrays with length() function
- **Collections**: User-defined struct-like types with automatic constructors
- **I/O Operations**: Built-in print functions and input reading

### Advanced Features
- **Compound Assignment Operators**: `+=`, `-=`, `*=`, `/=`, `%=`
- **Higher-Order Functions**: FUNC type for function pointers and callbacks
- **Function Composition**: Pass functions as parameters, return functions from functions
- **Type Safety**: Full semantic analysis with type checking
- **Block Scoping**: Proper variable scoping rules
- **Final Variables**: Immutable variable declarations

## Building & Running

### Prerequisites
- Java 21 or higher
- Gradle (included via wrapper)

### Build the Compiler
```bash
./gradlew build
```

### Run Tests
```bash
./gradlew test
```

### Compile a Source File
```bash
./gradlew run --args="source_file.lang -o output.class"
```

### Execute Generated Bytecode
```bash
java -cp output_directory class_name
```

## Language Syntax

### Variable Declaration
```lang
INT x = 5;
FLOAT pi = 3.14;
STRING name = "Hello";
BOOLEAN flag = true;
final INT constant = 42;
```

### Functions
```lang
def INT add(INT a, INT b) {
    return a + b;
}

def main() {
    INT result = add(5, 3);
    println(result);
}
```

### Collections (Structs)
```lang
coll Person {
    STRING name;
    INT age;
}

def main() {
    Person p = Person("Alice", 30);
    println(p.name);
}
```

### Arrays
```lang
def main() {
    INT[] numbers = INT ARRAY[5];
    numbers[0] = 10;
    println(numbers.length());
}
```

### Compound Operators
```lang
INT x = 5;
x += 3;  // x = x + 3
x *= 2;  // x = x * 2
x /= 4;  // x = x / 4
```

### Higher-Order Functions
```lang
def INT add(INT a, INT b) {
    return a + b;
}

def INT apply(FUNC f, INT x, INT y) {
    return f(x, y);  // Dynamic function call
}

def main() {
    INT result = apply(add, 5, 3);
    println(result);  // Outputs: 8
}
```

## Implementation Details

### Architecture
- **Lexer**: Tokenizes source code with support for multi-character operators and comments
- **Parser**: Recursive descent parser with 2-token lookahead
- **Semantic Analyzer**: Two-pass analysis (declaration collection, then validation)
- **Code Generator**: ASM library for JVM bytecode generation

### Key Technologies
- **ASM Library**: For bytecode generation
- **Java**: Implementation language
- **Gradle**: Build system
- **JUnit**: Testing framework
